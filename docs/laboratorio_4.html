Problema 6

  1. Calcule y grafique la transformada de Fourier de la función triángulo: $$x\left(t\right) = \Lambda\left(\frac{t}{2}\right)$$

  2. La integral que define la transformada de Fourier puede calcularse numéricamente, para cada valor de frecuencia, utilizando la suma de Riemman. Para subdominios de longitud T se tiene: $$X\left(f\right)=\sum_{N=-\infty}^{\infty}\Lambda\left(\frac{n T}{2}\right)\exp(-2\pi j f N T )T$$ Calcular para $T=0.8$ y para el rango de frecuencia de $0$ a $2$ con intervalos de $0.125$ ejecutando las siguientes sentencias en Matlab: 

                          T = 0.8;
                      n = [-2:2];
                      f = [0:0.125:2];
                      X = zeros(size(f)) ;
                      for i = 1: length(f)
                          X(i) = sum(T*triang(n*T/2).*exp(-j*2*pi*f(i)*n*T);
                      end
  3. Repetir para $T$ $10$ veces menor.

Nota: La función $\Lambda$ es la función triángulo $\Lambda\left(t\right)$ que Ud. debe implementar en Matlab®.
